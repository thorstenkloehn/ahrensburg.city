
# SSGs - Static Site Generators
SSGs ist eine Software, die statische Webseiten generiert. Sie sind besonders geejsonnen für Webseiten, die nicht oft geändert werden müssen. SSGs sind besonders schnell und sicher, da sie keine Datenbanken benötigen. Sie sind besonders geeignet für Blogs, Dokumentationen, Portfolios und andere Webseiten, die hauptsächlich aus Text und Bildern bestehen.
## Jekyll
Jekyll ist ein SSG, das in Ruby geschrieben ist. Es ist das beliebteste SSG und wird von GitHub Pages unterstützt. Jekyll ist sehr einfach zu benutzen und hat eine große Community. Es ist besonders geeignet für Blogs und kleine Webseiten.
## Hugo
Hugo ist ein SSG, das in Go geschrieben ist. Es ist sehr schnell und einfach zu benutzen. Hugo ist besonders geeignet für größere Webseiten und Webseiten, die viele Seiten haben.
## Gatsby
Gatsby ist ein SSG, das in JavaScript geschrieben ist. Es ist sehr flexibel und hat viele Plugins. Gatsby ist besonders geeignet für Webseiten, die viele interaktive Elemente haben.
## Next.js
Next.js ist ein SSG, das in JavaScript geschrieben ist. Es ist sehr flexibel und hat viele Plugins. Next.js ist besonders geeignet für Webseiten, die viele interaktive Elemente haben und die auf React basieren.
