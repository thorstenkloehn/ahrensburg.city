# Was ist Django
Django ist ein Web-Framework für Python, das die schnelle Entwicklung von Webseiten ermöglicht. Es ist Open Source und wurde ursprünglich von Adrian Holovaty und Simon Willison entwickelt. Django folgt dem Prinzip des Don't Repeat Yourself (DRY) und des Convention over Configuration. Das bedeutet, dass Entwickler:innen sich auf die Implementierung der Geschäftslogik konzentrieren können, während Django die Konfiguration und die Wiederholung von Code minimiert.

Django ist ein Full-Stack-Framework, das eine Vielzahl von Funktionen bietet, darunter:

- Ein ORM (Object-Relational Mapping) für die Interaktion mit der Datenbank
- Ein Template-System für die Erstellung von Webseiten
- Eine URL-Dispatcher für die Verwaltung von URLs
- Ein Admin-Interface für die Verwaltung von Inhalten
- Ein Authentifizierungs- und Autorisierungssystem für die Benutzerverwaltung
- Ein Formulare-System für die Verarbeitung von Benutzereingaben
- Ein Middleware-System für die Verarbeitung von Anfragen und Antworten
- Ein Cache-System für die Beschleunigung der Anwendung
- Ein Internationalisierungs- und Lokalisierungssystem für die Unterstützung mehrerer Sprachen

Django wird von einer großen und aktiven Community unterstützt und ist bei vielen Unternehmen und Organisationen beliebt. Es wird regelmäßig aktualisiert und verbessert, um mit den neuesten Entwicklungen in der Webentwicklung Schritt zu halten. Django wird von vielen bekannten Websites und Anwendungen verwendet, darunter Instagram, Pinterest, Disqus und Mozilla.


## Installation von Django 
Um Django zu installieren, benötigen Sie Python auf Ihrem System. Sie können Django mit dem Python-Paketmanager pip installieren. Führen Sie den folgenden Befehl aus, um Django zu installieren:

```bash
mkdir Django 
cd Django
python3 -m venv .venv
source .venv/bin/activate
pip install django
```
## Erstellen eines Django-Projekts
Nach der Installation von Django können Sie ein neues Django-Projekt mit dem Befehl `django-admin startproject` erstellen. Führen Sie den folgenden Befehl aus, um ein neues Django-Projekt mit dem Namen `myproject` zu erstellen:

```bash
django-admin startproject start .
python manage.py createsuperuser // Erstellen Sie einen Superuser
python manage.py migrate // Führen Sie die Migrationen aus
python3 manage.py runserver // Starten Sie den Entwicklungsserver
```
## Erstellen einer Django-App
In Django können Sie eine App erstellen, um spezifische Funktionen oder Anwendungen innerhalb Ihres Projekts zu organisieren. Führen Sie den folgenden Befehl aus, um eine neue App mit dem Namen `meineapp` zu erstellen:

```bash
python3 manage.py startapp meineapp
```
