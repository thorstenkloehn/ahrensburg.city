# Installieren github-pages

```bash
npm install -D gh-pages
```
## Installieren gh-pages in Next.js
    
    ```bash
    npm install gh-pages
    ```
## Konfigurieren package.json
    
    ```json
    "scripts": {
        "deploy": "npm run build && gh-pages -d out"
    }
    ```
## Deployen
    
    ```bash
    npm run deploy
    ```
## Konfigurieren next.config.js
    
    ```js
    module.exports = {
        basePath: '/my-app',
        assetPrefix: '/my-app/',
    }
    ```
