# Dokument
## Drogon C++ Framework installieren
### Systemanforderungen
- Das Linux-Kernel sollte nicht niedriger als 2.6.9 sein, 64-Bit-Version;
- Die gcc-Version sollte nicht weniger als 5.4.0 sein, es wird empfohlen, Version 11 oder höher zu verwenden;
- Verwenden Sie cmake als Build-Tool, und die cmake-Version sollte nicht weniger als 3.5 sein;
- Verwenden Sie git als Versionsverwaltungstool;

## Ubuntu 18.04
### Umgebung einrichten
- `git`: Ein verteiltes Versionskontrollsystem, das von Entwicklern verwendet wird, um Änderungen am Quellcode zu verfolgen und zu verwalten.
- `build-essential`: Ein Paket in Debian-basierten Systemen, das Referenzversionen von `gcc`, `g++` und anderen Entwicklungstools enthält. Es ist eine grundlegende Anforderung für das Kompilieren von C- und C++-Programmen.
- `cmake`: Ein Open-Source-Tool zur Generierung von Makefiles oder Projekten für verschiedene Plattformen. Es wird verwendet, um den Build-Prozess in einer Betriebssystem-unabhängigen Weise zu steuern.

```bash
 sudo apt install git
sudo apt install build-essential
sudo apt install cmake
```

## jsoncpp
 Eine Open-Source-C++-Bibliothek, die Funktionen zum Parsen und Serialisieren von JSON-Daten bietet. Sie ist einfach zu verwenden und bietet eine flexible API, die es Entwicklern ermöglicht, JSON-Daten mit C++-Code zu arbeiten.

```bash
sudo apt install libjsoncpp-dev
```
## uuid
Steht für "Universally Unique Identifier". Es handelt sich dabei um eine Reihe von 128 Bits, die eindeutig sind. UUIDs werden in der Softwareentwicklung häufig verwendet, um Datenbanken und Dateisysteme eindeutig zu identifizieren.
```bash 
sudo apt install uuid-dev
```
## zlib
Eine Open-Source-Bibliothek, die Funktionen zum Komprimieren und Dekomprimieren von Daten bereitstellt. Sie ist einfach zu verwenden und bietet eine flexible API, die es Entwicklern ermöglicht, Daten mit C++-Code zu komprimieren und zu dekomprimieren.
```bash
sudo apt install zlib1g-dev
```
## openssl
Eine Open-Source-Bibliothek, die Funktionen zum Verschlüsseln und Entschlüsseln von Daten bereitstellt. Sie ist einfach zu verwenden und bietet eine flexible API, die es Entwicklern ermöglicht, Daten mit C++-Code zu verschlüsseln und zu entschlüsseln.
```bash
sudo apt install openssl
sudo apt install libssl-dev
```
## PostgreSQL
Ein Open-Source-Objekt-Relationales Datenbankmanagementsystem (ORDBMS), das auf der SQL-Sprache basiert. Es ist einfach zu verwenden und bietet eine flexible API, die es Entwicklern ermöglicht, Datenbanken mit C++-Code zu erstellen, zu verwalten und zu abfragen.
```bash
sudo apt-get install postgresql-server-dev-all
```
