# Dokument
## Drogon C++ Framework installieren
### Systemanforderungen
- Das Linux-Kernel sollte nicht niedriger als 2.6.9 sein, 64-Bit-Version;
- Die gcc-Version sollte nicht weniger als 5.4.0 sein, es wird empfohlen, Version 11 oder höher zu verwenden;
- Verwenden Sie cmake als Build-Tool, und die cmake-Version sollte nicht weniger als 3.5 sein;
- Verwenden Sie git als Versionsverwaltungstool;

## Ubuntu 18.04
### Umgebung einrichten

- `sudo apt install git`: Dieser Befehl installiert `git`, ein Versionskontrollsystem, das von Entwicklern verwendet wird, um Änderungen am Quellcode zu verfolgen.
- `sudo apt install gcc`: Dieser Befehl installiert `gcc`, den GNU Compiler Collection, der zum Kompilieren von C- und C++-Programmen verwendet wird.
- `sudo apt install g++`: Dieser Befehl installiert `g++`, einen Teil der GNU Compiler Collection, der speziell zum Kompilieren von C++-Programmen verwendet wird.
- `sudo apt install cmake`: Dieser Befehl installiert `cmake`, ein Open-Source-Tool, das zum Generieren von Makefiles oder Projekten für verschiedene Plattformen verwendet wird.
```bash
 sudo apt install git
sudo apt install gcc
sudo apt install g++
sudo apt install cmake
```
