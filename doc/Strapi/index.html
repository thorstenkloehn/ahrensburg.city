Strapi Installation

Dieser Abschnitt beschreibt die Installation von Strapi, einem Open-Source-Headless-CMS, mit PostgreSQL als Datenbank.

Führen Sie die folgenden Befehle in Ihrem Terminal aus:

  1. sudo -u postgres -i: Dieser Befehl wechselt zu dem Benutzer postgres. sudo -u wird verwendet, um Befehle mit den Berechtigungen eines anderen Benutzers auszuführen, und -i startet eine neue Shell mit diesem Benutzer.

  2. createuser thorsten: Dieser Befehl erstellt einen neuen PostgreSQL-Benutzer namens "thorsten". Dieser Befehl wird normalerweise von einem Superuser ausgeführt, in diesem Fall "postgres".

  3. createdb -E UTF8 -O thorsten strapi: Dieser Befehl erstellt eine neue PostgreSQL-Datenbank mit dem Namen "strapi". Die Option -E UTF8 setzt das Encoding der Datenbank auf UTF8, und die Option -O thorsten setzt den Eigentümer der Datenbank auf den Benutzer "thorsten". Nachdem Sie diese Befehle ausgeführt haben, sollten Sie eine neue PostgreSQL-Datenbank namens "strapi" haben, die von dem Benutzer "thorsten" verwaltet wird. Sie können nun Strapi installieren und es so konfigurieren, dass es diese Datenbank verwendet

  4. npx create-strapi-app@latest strapi: Dieser Befehl erstellt eine neue Strapi-App mit dem Namen "strapi" und verwendet den postgresql-Template. Sie können auch andere Datenbanken wie MySQL oder SQLite verwenden, indem Sie das entsprechende Template angeben.

  5. cd strapi: Wechseln Sie in das Verzeichnis Ihrer neuen Strapi-App.

  6. npm run develop: Starten Sie die Strapi-App im Entwicklungsmodus. Sie können nun auf http://localhost:1337 zugreifen, um die Strapi-Oberfläche zu öffnen und Ihre Inhalte zu verwalten.
