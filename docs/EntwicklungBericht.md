## Entwicklung Bericht

### Webpack 

Webpack wird verwendet, um JavaScript-Module zu kompilieren. Nach der Installation können Sie mit dem Webpack entweder über dessen CLI oder API interagieren . Wenn Sie Webpack noch nicht kennen, lesen Sie sich bitte die Kernkonzepte und diesen Vergleich durch, um zu erfahren, warum Sie es möglicherweise gegenüber anderen Tools in der Community verwenden.

#### Grundeinstellung
Lassen Sie uns zuerst ein Verzeichnis erstellen, npm initialisieren, webpack lokal installieren und das webpack-cli(das Tool zum Ausführen von webpack auf der Befehlszeile) installieren:

```
mkdir ahrensburg.city
cd ahrensburg.city
git init
npm init -y
npm install webpack webpack-cli --save-dev
npm install --save lodash
```
